# 📌 KeenKeeper

To show the live demo <a href="https://keen-keeper-eight-sigma.vercel.app/">Click Here</a>

**KeenKeeper** is a modern web application designed to help you maintain and nurture your friendships. It allows users to track interactions, monitor relationship status, and visualize communication patterns over time.

## 🛠️ Technologies Used

- Next.js (App Router)
- React.js
- Tailwind CSS
- React Context API (State Management)
- Recharts (Data Visualization)
- React Icons
- React Toastify
- Next/Image & Next/Link
- JavaScript (ES6+)

## 🚀 Features

- 🏠 Clean and modern Home page with banner and call-to-action
- 👥 View all friends in a responsive grid layout
- 📄 Detailed Friend Profile with bio, tags, and contact info
- 📞 Quick interaction options (Call, Text, Video)
- ⏱️ Automatic timeline tracking for every interaction
- 🔍 Filter timeline by interaction type (All, Text, Call, Video)
- 📊 Visual analytics using Pie Chart for interaction breakdown
- 📈 Real-time friendship status summary (On-track, Overdue, Total)
- 🔔 Toast notifications for user actions
- ⚠️ Custom error handling page with retry option
- 🚫 Custom 404 Not Found page
- ⏳ Loading spinner for better UX
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🔄 Global state management using React Context API
- 🧩 Reusable UI components for maintainability
- 🎨 Smooth animations and hover effects for better user experience
